== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `responsive-video-ultimate` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.

Shortcodes

[youtube]youtube_video_id_here[/youtube]

[vimeo]vimeo_video_id_here[/vimeo]

 

[dailymotion]dailymotion_video_id_here[/dailymotion]

[funnyordie]funnyordie_video_id_here[/funnyordie]

[blip_tv]blip_tv_video_id_here[/blip_tv]

[ustream_tv]ustream_tv_video_id_here[/ustream_tv]
